# Russia-Ukraine Conflict Matrix

Ez a projekt az orosz–ukrán háború napi monitorozására készült.

## Funkciók
- hírek begyűjtése RSS forrásokból
- konfliktusintenzitás és trend követése
- napi riport generálása
- dashboard adatok előállítása
- GitHub Actions automatikus futtatás

## Fő workflow
1. hírek begyűjtése
2. hírek pontozása
3. konfliktustörténet frissítése
4. riport generálása
5. grafikon és dashboard adat előállítása

## Mappák
- `src/` – Python szkriptek
- `data/` – nyers és feldolgozott adatok
- `reports/` – generált riportok
- `docs/` – dashboardhoz kapcsolódó fájlok
